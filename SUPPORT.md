# Support / 支持

- Usage questions and troubleshooting: open a GitHub Discussion or use the
  Question issue template if Discussions are unavailable.
- Reproducible defects: use the Bug Report template.
- Feature proposals: use the Feature Request template.
- Security vulnerabilities: follow [SECURITY.md](SECURITY.md), never a public issue.

This is a community-maintained project without a commercial support SLA.
Questions about the upstream TencentDB Agent Memory project should be directed
to its own community channels.

使用咨询优先提交 GitHub Discussion；可复现缺陷请使用 Bug 模板；功能建议请使用 Feature
模板；安全问题必须按 [SECURITY.md](SECURITY.md) 私密报告。本项目由社区独立维护，不提供
商业 SLA，上游项目问题请前往上游社区反馈。
